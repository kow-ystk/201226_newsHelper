カレーを作る

このページについて

スパイスカレーを作ります。
市販のカレールーを使わず、自分でスパイスを調合するので楽しいですよ。

調理過程

  • 玉ねぎを炒める

    ピュアオリーブオイルで、みじん切りにした玉ねぎを炒めます。
    きつね色くらいまで色が変わるのが目安です。

  • 他の野菜も炒める

    他の野菜も投入します。
    定番である人参の他に、トマトやかぼちゃを入れていますがお好みです。
    トマトを入れた場合は、後の工程で水をあまり入れなくて良いかもしれません。

  • 鶏肉も炒める

    鶏肉も投入します。
    蓋をして、火を十分に通すように気をつけてください。

  • スパイスを投入

    具材が十分に炒まったところでスパイスを投入します。
    クミン、ターメリック、コリアンダー、カイエンペッパーの4種類があれば十分です。
    コリアンダーの入れすぎは、爽やかな感じが出すぎてしまうので気をつけてください。

完成

スパイスカレーの完成です!!
カレールーを使わずに自分で作った達成感は格別です。
慣れてきたら、カルダモンやクローブなど他のスパイスも足してみたりしてオリジナルの調合を楽しみましょう！
